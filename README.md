Umar Shaikh - Interactive Portfolio
A high-performance, immersive portfolio website designed to showcase full-stack development skills through advanced motion design and visual storytelling.

About
This project represents a modern approach to digital identity, merging technical depth with high-end aesthetics. The design focuses on bold typography, fluid interactivity, and a minimalist dark mode interface that allows the content to stand out. It is built to offer a seamless experience across all devices, from large desktop screens to mobile phones.

Key Features
Dynamic Neon Theming: The site features a randomization engine that selects one of ten distinct neon accent colors every time the page loads, giving every visitor a unique visual experience.

Custom Preloader: A bespoke, SVG-based radioactive symbol loader that fills dynamically from 0% to 100%, setting the tone for the site's performance-focused architecture.

Fluid Motion: Integrated smooth scrolling (Lenis) and physics-based animations create a "heavy," premium feel to the navigation.

Interactive Typography: Text elements feature curtain-reveal animations and magnetic hover effects, powered by Framer Motion.

Responsive Design: A mobile-first architecture ensuring the massive typography and layout scale perfectly to any screen size.

Technologies Used
Core: React (Vite)

Styling: Tailwind CSS

Animation: Framer Motion

Scroll Physics: Lenis

Fonts: Anton (Display) & Manrope (Body)
